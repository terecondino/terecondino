# Upload and publish on GitHub Pages

## 1. Upload the website

1. Extract `terecondino-portfolio.zip` on your computer.
2. Open <https://github.com/terecondino/terecondino>.
3. Select **Add file → Upload files**.
4. Upload these files to the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `TERESA-LINKEDIN.webp`
   - `README.md`
5. Enter a commit message such as `Add Teresa's QA portfolio` and select **Commit changes**.

Do not upload the ZIP file itself. GitHub needs the extracted files.

## 2. Enable GitHub Pages

1. Open the repository's **Settings**.
2. Select **Pages** under **Code and automation**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch **main**, folder **/(root)**, and click **Save**.
5. Wait a few minutes for deployment.

The website will be available at:

<https://terecondino.github.io/terecondino/>
